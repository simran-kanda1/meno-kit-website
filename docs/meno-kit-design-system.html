Brand feel

Warm, editorial, lavender-toned. Cream backgrounds, serif headlines, soft purple accents. Calm and premium — not clinical, not startup-neon.

  • Logo: circular lavender “m” with botanical detail on black (menokit-logo.png)
  • Wordmark: MENO-KIT — uppercase serif, wide letter-spacing
  • Tagline: Menopause support that adapts with you.

Splash screen

Shown once per browser session before the site loads (~1s hold, then 450ms fade). Skipped if user prefers reduced motion.

Background: #F5F0E8 + subtle mesh gradient (4 blobs at 80% opacity)

Logo: menokit-logo.png, centered — 112×112px mobile, 128×128px desktop

z-index: 200 (full-screen overlay)
